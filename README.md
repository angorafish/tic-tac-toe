Tic-Tac-Toe Game

Description: A web-based Tic-Tac-Toe game where two players compete by marking X and O on a 3x3 grid. The game tracks scores across multiple sessions using localStorage and includes a simple AI opponent.

Problem it Solves: Offers a fun and easy-to-understand game that demonstrates basic web development and JavaScript skills.
Target Audience: Casual gamers and developers learning JavaScript.

Features:

	•	AI implementation for a single-player experience.
	•	Score tracking across sessions using localStorage.
	•	Game and score reset functionality.

Technologies Used

	•	HTML: Structure and layout.
	•	CSS: Styling and visual design.
	•	JavaScript: Game logic, score tracking, and DOM manipulation.
	•	localStorage: Persisting scores across sessions.

Challenges and Solutions

Challenge: Implementing AI to provide a fair opponent. Initially, a trained AI was too challenging, making it impossible to win.
Solution: Simplified the AI by implementing a basic algorithm that makes random valid moves, ensuring a more balanced game.

User Guide / FAQ

How to Play:

	•	Click on an empty square on the game board to place your mark (X). The AI plays as O.
	•	The game automatically tracks your score.

How to Reset the Game: Use the “Restart Game” button to reset the board.
How to Reset Scores: Use the “Reset Scores” button to clear the scores.

Deployment Instructions

	1.	Clone the GitHub repository.
	2.	Open index.html in a web browser (using a live server is recommended) to play the game.
